# SERMHS_SchoolGarden  
Southeast Raleigh Magnet High School Website  

## Getting Started  

Follow these steps to set up and run the project:  

1. **Install Dependencies**  
   ```shell  
   npm install  
   ```  

2. **Run the Application**  
   - For a development environment:  
     ```shell  
     npm run dev  
     ```  
   - For a production environment:  
     ```shell  
     npm run start  
     ```
